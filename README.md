Este projeto é um sistema web de gerenciamento de academia desenvolvido com Django e interface utilizando Bootstrap.

A aplicação permite o cadastro e gerenciamento de alunos (CRUD), além do controle de entrada e saída, registrando automaticamente data e hora de acesso.
Também possui um módulo de cálculo de IMC, baseado nos critérios da Organização Mundial de Saúde (OMS), com sugestões de exercícios.

O sistema inclui ainda um módulo de pagamento de mensalidades, que define automaticamente se o aluno está ativo ou inadimplente, além de um dashboard interativo com
indicadores como total de alunos e presença atual.

De forma geral, o projeto simula um sistema real de gestão de academia, integrando controle de usuários, frequência, saúde e financeiro em uma única aplicação.
